## day01 和 day 02 中的笔记

#### 汇编语言
  - db  : data byte 向文件中写入一个字节(8Bit)
  - dw  : data word 向文件中写入一个字(16Bit)
  - dd  : data  double  向文件中写入两个字（32Bit）
  利用上面的命令写入数据，表示后面的数据以多长的长度来进行存储。eg: db "HELLOIPL"，则是以一个字节来
  存储没一个字符。dw  512 : 则表示以两个字节来存储512这个数据。 db  0xeb, 0x4e,0x90:则是存储了三个
  字节
  - times : 重复执行某一个命令
  - $ : 当前行的地址
  - $$  : 当前section的基地址，
